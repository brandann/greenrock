Skyda

Skyda is still in early development.

Skyda is a Top Down RPG game that is easy to learn and play, yet the gameplay and story is progressive and expansive. The player must find and conquer dungeons in an open world, collecting items and abilities, all while managing health surviving the night.

World
The story takes place in a random world called Skyda that is filled with an endless number of dugeons for the player to defeat.

Whats Unique?
Skyda takes elements from:
Zelda (Link to the past): Art, Gameplay, Dungeons.
Skyrim: Expansive world, open gameplay, crafting?, magic?
Minecraft: Random world, Day/Night, Neverending

Goals
The game will be designed in three phases.
The first phase being the vertical slice and alpha (tentative) with all/most game mechanics developed.
The second phase utilizing Agile processes to create alpha (tentative), beta, release, and updates
The third phase is to update the game to have a new feature (online multiplayer) that should effect all the code of the game, giving the developers experience revisiting and refactoring code to include new features from a stake holder.
